This is a very simple plugin to display a random quotes in your posts or widgets

== Description ==

Simple Quote Rotator allows you to present your quotes to your WordPress visitors.
Learn more or connect with us:
http://www.kulturystyka.sklep.pl/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/simple-quote-rotator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add to any of your post shortcode [simple-quote-rotator] to display random quotes.

== Frequently Asked Questions ==

= What is Simple Quote Rotator? =
Simple Quote Rotator allows you to present your quotes to your WordPress visitors.

= How to add my own quotes? =

Just simply open with text editor file quotes.xml in plugin folder and add your qutoes.

= How to display more or less than 19 quotes? =

Edit file simple-quote-rotator.php and change number "19" to greater or smaller depending on the number of your quotes.
